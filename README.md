## Day 1 Git Terms
Git is a memory card for code.
GitHub is a website to publish your saved code onto the website.

--> Respository = Folder
--> Push = pushing your saved comits to GitHub
--> Pull = to stay updated with changes in the GitHub respository

--> Branches = making changes to the code, without disrupting the main branch.
    --> Merging branches can combine different branches, which is optional.
    --> Pull Request = To notify members about branches to suggest a merge or discuss options.
